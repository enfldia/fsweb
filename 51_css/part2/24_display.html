블록레벨 요소 vs 인라인 요소

블록레벨 요소

  • display:block; (단, li는 display:list-item)
  • 줄바꿈o:위에서 아래로 배치(세로 배치)
  • html, body, div, h1~h6, p, ul, li, ...
  • 기본값이 width:100%; height:콘텐츠크기; border: none
  • width, height, margin, padding, border를 사용자가 CSS로 지정할 수 있다.

인라인 요소

  • display:inline;
  • 줄바꿈x:왼쪽에서 오른쪽으로 배치(가로 배치)
  • span, img, a, em, b, i, label, ...
  • 기본값이 width:콘텐츠크기; height:콘텐츠크기; border: none
  • width와 height는 지정할 수 없다. (단, 이미지는 예외)
  • margin은 좌우만 적용된다.
  • padding, border는 모두 적용된다.

인라인 요소(디스플레이 인라인 블록)

  • display:inline-block;
  • 줄바꿈x:왼쪽에서 오른쪽으로 배치(가로 배치)
  • button, input, ...
  • width, height, margin, padding, border 사용자가 CSS로 지정할 수 있다.
  • button은 box-sizing:border-box; 이다.
